# G4K Exhibition Finder

Interactive tool for evaluating church conferences and exhibitions where Givt4Kids (G4K) should exhibit.

Built and maintained with Claude (Cowork mode).

## Live URL

https://sivanoort-coder.github.io/g4k-exhibition-finder/

## Stack
- React 18 (via CDN)
- Babel standalone (JSX in browser)  
- GitHub Pages (static hosting, zero build step)
